graphdb-javascript-driver (RDF4J.js)

A GraphDB data access library written in JavaScript to be used in Node.js and/or web browser environment.

Installation

npm install

Node.js

Make sure you have Node.js and Node Package Manager (npm) installed before start working with the library.

npm install --save rdf4js

Dev build

npm run build:dev

Production build

npm run build

Run tests

npm run test

Run lint

The library uses Google style in conjunction with ESLint's recommended ruleset.

npm run lint
